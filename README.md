# EV-Analysis-Project
EV Analysis Project ⚡
📌 Overview

This project focuses on analyzing Electric Vehicle (EV) data to identify trends in performance, pricing, and battery efficiency. Using Python, I processed datasets from both Indian and global markets to visualize the growth and technical specifications of modern EVs.
🛠️ Tech Stack

Language: Python
Libraries: Pandas (for data filtering), Matplotlib & Seaborn (for generating graphs)
Tools: GitHub for version control

📂 Project Structure

Data-Filter.Py: Cleans the raw CSV files and removes unnecessary information.
EV-Visualization.Py: Generates the graphs and charts used for analysis.
DB-Performance-Test.py: A script to check the efficiency of data processing.
EVIndia.csv: Dataset containing EV details specific to the Indian market.
